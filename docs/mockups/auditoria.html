Como funciona (sem custo extra): cada ocorrência já guarda um histórico interno de tudo que aconteceu com ela. Esta tela só junta tudo numa linha do tempo única, ordenada do mais recente, com busca e filtros. Vale retroativo — já mostra o que está registrado hoje.
  • Quem vê: só Admin e RH (informação sensível).
  • Clicar numa linha abre a ocorrência correspondente.
  • Onde fica: sugiro um item novo no menu — "Auditoria" — ou uma aba dentro de Configurações. Você escolhe.
  • Único ponto cego: ocorrência excluída some com o histórico junto. Se quiser registrar exclusões também, dá pra gravar um log leve só pra isso (me avise).
