Fluxos seguros — undo, diff de import e conferência guiada

Estes são os fluxos de maior risco trabalhista e financeiro do app. A diretriz é simples: toda ação destrutiva ganha uma rede de segurança — desfazer no toast para o que é reversível, confirmação com comparativo para o que não é — e ninguém decide no escuro: a conferência explica a consequência de cada ação, o import mostra o diff antes de substituir, e quem cria a ocorrência vê o contexto do funcionário na hora.
