Proposta · abertura no desktop

Esqueleto de abertura, versão desktop

Hoje, no desktop, o esqueleto de boot é o mesmo cartão estreito do celular, perdido no meio da tela. A proposta é uma silhueta que imita o layout largo do painel: uma barra lateral à esquerda e o conteúdo em grade à direita, pra dar a sensação de que o dashboard já está tomando forma. Mesma cor corrigida e o "Carregando...".

4 Desktop  · silhueta de sidebar + grade larga (imita o painel do gestor)
Carregando...

Como fica. No celular continua o esqueleto atual (avatar + atalhos + cards, agora com contraste corrigido). No desktop entra essa silhueta larga só a partir de 901px de largura, some assim que o app carrega, igual à mobile.

Um ponto pra você decidir: a abertura acontece antes de o app saber se você é gestor ou colaborador. Essa silhueta imita o painel do gestor (com sidebar). Pro colaborador no desktop o conteúdo é mais enxuto e centralizado, sem sidebar grande, então pra ele essa silhueta larga fica um pouco "a mais". Como quase todo mundo no desktop é gestor, eu iria com essa mesmo; mas se preferir, dá pra fazer uma silhueta neutra (uma coluna larga centralizada, sem sidebar) que serve pros dois. Me diz qual você quer.
